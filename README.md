# FullStackOpen Part 8 Exercises

Submission for FullStackOpen [Part 8: GraphQL](https://fullstackopen.com/en/part8) exercises

## Content covered

- [x] using Apollo Studio
- [x] creating, using and updating a context
- [x] implementing a GraphQL server via Apollo Server
  - startStandAloneServer
  - use expressMiddleware
  - resolvers and typeDefs
- [x] using GraphQL with React via Apollo Client (React):
  - queries
  - mutations
  - fragments
  - subscriptions
  - types, enums
- [x] updating cache in client

## To Run

1. Clone the repo
2. Install all packages
   ```sh
   npm install
   cd ./react-graphql
   npm install
   ```
3. Use your mongodb URI in .env
4. Run locally
   ```sh
    npm run dev
    cd ./react-graphql
    npm run dev
   ```
